## 1. JSON转换和格式化

### 功能说明

用于格式化和验证JSON数据，可以将压缩的JSON字符串转换为格式化的、易读的形式。

### 使用方法

1. 在输入框中粘贴或输入JSON字符串
2. 点击"转换"按钮
3. 格式化后的JSON显示在输出框中
4. 点击"复制结果"复制到剪贴板

### 示例

**输入**:
```json
{"name":"张三","age":30,"city":"北京","hobbies":["读书","旅行"]}
```

**输出**:
```json
{
  "name": "张三",
  "age": 30,
  "city": "北京",
  "hobbies": [
    "读书",
    "旅行"
  ]
}
```

### 注意事项

- 输入的JSON必须是有效的JSON格式
- 如果JSON包含转义字符，工具会自动处理
- 格式化使用2个空格缩进

## 2. XML格式化

### 功能说明

用于格式化和美化XML数据，使XML结构更加清晰易读。

### 使用方法

1. 在输入框中输入XML字符串
2. 点击"转换"按钮
3. 格式化后的XML显示在输出框中

### 示例

**输入**:
```xml
<root><item id="1"><name>测试</name><value>123</value></item></root>
```

**输出**:
```xml
<root>
  <item id="1">
    <name>测试</name>
    <value>123</value>
  </item>
</root>
```

### 注意事项

- 输入的XML必须是格式良好的（well-formed）
- 工具会自动添加适当的缩进和换行

## 3. 加密/解密文本

### 功能说明

提供多种加密算法的文本加密和解密功能，包括凯撒密码和Base64编码。

### 使用方法

1. 在输入框中输入要加密或解密的文本
2. 选择加密方式（凯撒密码/Base64）
3. 输入必要的参数（如凯撒密码的位移量）
4. 点击"加密"或"解密"按钮
5. 结果显示在输出框中

### 加密方式

#### 凯撒密码（Caesar Cipher）

- **位移量**: 输入一个数字（1-25），表示字母偏移的位置
- **加密**: 将每个字母向前移动指定位置
- **解密**: 将每个字母向后移动指定位置

#### Base64编码

- **编码**: 将文本转换为Base64编码
- **解码**: 将Base64编码还原为原始文本

### 示例

**凯撒密码加密（位移量=3）**:
- 输入: `HELLO`
- 输出: `KHOOR`

**Base64编码**:
- 输入: `Hello World`
- 输出: `SGVsbG8gV29ybGQ=`

3. 解码后的文本显示在输出框中

**文本转Base64**:
- 输入: `开发工具`
- 输出: `5byA5Y+R5Lqk5a2m`

**Base64转文本**:
- 输入: `SGVsbG8gV29ybGQ=`

## 5. Base64转图片

### 功能说明

将Base64编码的图片数据转换为图片预览，并支持下载。

### 使用方法

1. 在输入框中粘贴Base64编码的图片数据
   - 格式: `data:image/png;base64,iVBORw0KGgoAAAANS...`
   - 或纯Base64字符串: `iVBORw0KGgoAAAANS...`
2. 点击"转换"按钮
3. 图片预览显示在输出区域
4. 点击"下载图片"保存图片文件

### 支持的图片格式

- PNG
- JPEG/JPG
- GIF
- WebP
- BMP

### 注意事项

- Base64字符串应该以 `data:image/...;base64,` 开头，或者工具会自动添加
- 如果Base64字符串无效，会显示错误信息

## 6. 大小写转换

### 功能说明

在不同的大小写格式之间转换文本。

### 转换类型

1. **转换为大写** - 所有字母转为大写
2. **转换为小写** - 所有字母转为小写
3. **首字母大写** - 每个单词的首字母大写（标题格式）
4. **句首大写** - 每句话的首字母大写（句子格式）

### 使用方法

1. 在输入框中输入文本
2. 选择转换类型
3. 点击相应的转换按钮
4. 转换结果显示在输出框中

### 示例

**输入**: `hello world. this is a test.`

- **大写**: `HELLO WORLD. THIS IS A TEST.`
- **小写**: `hello world. this is a test.`
- **首字母大写**: `Hello World. This Is A Test.`
- **句首大写**: `Hello world. This is a test.`

## 8. 文本到Unicode

### 功能说明

实现文本与Unicode编码之间的双向转换，支持两种格式。

### Unicode格式

1. **\u格式** - 标准Unicode格式，如 `\u4e2d`
2. **U+格式** - 十六进制格式，如 `U+4E2D`

### 使用方法

1. 在输入框中输入文本或Unicode编码
2. 选择Unicode格式（\u或U+）
3. 点击"转换为Unicode"或"转换为文本"

### 示例

**文本转Unicode (\u格式)**:
- 输入: `中国`
- 输出: `\u4e2d\u56fd`

**Unicode转文本**:
- 输入: `\u0048\u0065\u006c\u006c\u006f`
- 输出: `Hello`

## 9. JSON差异比较

### 功能说明

比较两个JSON对象之间的差异，识别添加、删除、修改等变化。

### 使用方法

1. 在"JSON 1"输入框中输入第一个JSON对象
2. 在"JSON 2"输入框中输入第二个JSON对象
3. 点击"比较"按钮
4. 差异结果以高亮形式显示

### 差异类型

- **添加** - 在第二个JSON中存在但第一个中不存在的字段
- **删除** - 在第一个JSON中存在但第二个中不存在的字段
- **修改** - 值发生变化的字段
- **类型改变** - 字段类型发生变化（如字符串变为数字）

### 示例

**JSON 1**:
```json
{
  "name": "张三",
  "age": 30
}
```

**JSON 2**:
```json
{
  "name": "李四",
  "age": 30,
  "city": "北京"
}
```

**差异**:
- `name`: 值改变（"张三" → "李四"）
- `city`: 添加（"北京"）

## 12. ASCII艺术字生成器

### 功能说明

将普通文本转换为ASCII艺术字体，生成美观的文本艺术效果。

### 字体类型

1. **标准字体** - 经典ASCII艺术字体
2. **方块字体** - 使用方块字符的字体
3. **简单字体** - 简化的ASCII字体

### 使用方法

1. 在输入框中输入文本（通常支持字母和数字）
2. 选择字体类型
3. 点击"生成"按钮
4. ASCII艺术字显示在输出框中

### 示例

**输入**: `HELLO`

**输出** (标准字体):
```
 _   _      _ _        _ 
| | | |    | | |      | |
| |_| | ___| | | ___  | |
|  _  |/ _ \ | |/ _ \ | |
| | | |  __/ | | (_) |_|
|_| |_|\___|_|_|\___/(_)
```

### 注意事项

- 某些字符可能不支持所有字体
- 输出结果可能较长，建议使用较小的文本

## 14. JSON转JavaBean

### 功能说明

根据JSON对象自动生成JavaBean类代码，包括字段定义和getter/setter方法。

### 使用方法

1. 在输入框中输入JSON对象
2. 输入JavaBean类名
3. 点击"转换"按钮
4. Java代码显示在输出框中
5. 复制代码到Java项目中使用

### 生成内容

- 类的包声明（可选）
- 类的字段定义（使用适当的Java类型）
- Getter方法
- Setter方法
- 支持嵌套对象（生成内部类）

### 示例

**输入JSON**:
```json
{
  "name": "张三",
  "age": 30,
  "email": "zhangsan@example.com"
}
```

**输出Java代码**:
```java
public class User {
    private String name;
    private Integer age;
    private String email;

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }

    // ... 其他getter/setter
}
```

### 类型映射

- `string` → `String`
- `number` → `Integer` 或 `Double`（根据数值）
- `boolean` → `Boolean`
- `array` → `List<T>`
- `object` → 内部类

### 注意事项

- 生成的代码可能需要手动调整
- 嵌套对象会生成内部类
- 数组类型需要指定泛型参数
